# 🔐 Login System (Java Servlet + JSP + JPA)

## 🚀 Features
- User Registration
- Login Authentication
- Session Management
- Logout

## 🛠 Tech Stack
- Java (Servlet, JSP)
- Hibernate (JPA)
- MySQL
- Apache Tomcat

## ▶️ How to Run
1. Import as Maven project
2. Configure DB in persistence.xml
3. Run on Tomcat
4. Open /login.jsp
